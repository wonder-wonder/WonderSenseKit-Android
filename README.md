# WonderSenseKit

This is Android SDK for WonderSense.


## Installation
### Maven

```txt
repositories {
    maven { url "https://github.com/wonder-wonder/ws-ble-v1-android-sdk/raw/develop/BLETest/repository/" }
}

dependencies {
    implementation 'xyz.wonder_wonder.android:wondersensekit:1.0.2'
}
```
